What is Ratpack?

Ratpack apps are lightweight, fast, composable with other tools & libraries, easy to test… and enjoyable to develop.

Ratpack is implemented in Java, designed with Groovy in mind. Apps can be implemented as a single Groovy script, or can scale up to the large with your build tool of choice. We prefer Gradle and provide tight integration.

Ratpack is built on Netty and is designed for asynchronous request processing. Synchronous, blocking, style can also be used. Ratpack is not a J2EE solution. Deployments are typically as executable JARs.

Getting Started

Create a Groovy script on your filesystem with: 

@Grab("org.ratpack-framework:ratpack-groovy:0.9.0")
import static org.ratpackframework.groovy.RatpackScript.ratpack

ratpack {
  handlers {
    get {
      response.send "Hello world!"
    }
  }
}

That's just the tip of the iceberg.

Check out the Ratpack Manual for more possibilities.
